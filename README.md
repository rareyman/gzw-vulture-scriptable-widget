# gzw-vulture-scriptable-widget
Where is Vulture?

## About
This adds a Scriptable widget to your iOS home screen that tells you where Vulure (a vendor in the game Gray Zone Warfare) is.

## Installation
- Install Scriptable on iOS App Store on iPhone/iPad
- Copy code from `Where is Vulture?.js`
- In Scriptable, create a new script and name it "Where is Vulture?"
- Paste in code
- Choose Done
- Add a Scriptable medium widget
- Long press the widget and choose Edit
- Select the Where is Vulture? script
- Optional: Select "Run Script" on "When Interacting"
